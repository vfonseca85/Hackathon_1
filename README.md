# A Diary App - Pandora's Box

## Description
A project where the goal is to create a Bridget Jones Diary which allows you to see, create, update and delete diary entries.

## Installation and Usage
### Installation
`npm install -D` or `npm i -D`

### Usage
1. `npm run setup-db`
2. `npm run dev`

## Technologies
1. Javascript 
2. Express
3. SQL
4. HTML
5. CSS
6. ElephantSQL
